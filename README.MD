# Ordinly - Landing Page

Projeto de uma landing page para um app fictício de organização de tarefas.

## 🚀 Tecnologias
- HTML5
- CSS3 (Flexbox & Grid)
- Mobile First

## 🎯 Objetivo
Praticar construção de layouts reais, responsividade e organização de código.

## 📱 Funcionalidades
- Layout responsivo (mobile → desktop)
- Seções de benefícios e recursos
- Call to Action (CTA)
- Estrutura semântica
